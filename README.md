## ✨ Особенности

### 🎯 **Основной функционал**
- ✅ **Автоматическая продажа Telegram Stars** (50-2500 ⭐)
- ✅ **Автоматическая продажа Telegram Premium** (3-12 месяцев)
- ✅ **Интеграция с Fragment API** для мгновенной отправки
- ✅ **Оплата через CloudTips** (российская платежная система)
- ✅ **Полностью кнопочный интерфейс** - никаких команд!

### 🔧 **Технические возможности**
- 💾 **SQLite база данных** для хранения пользователей и платежей
- 👥 **Реферальная система** (10% от покупок рефералов)
- 📊 **Подробная статистика** для пользователей и админа
- ⏰ **Автоматическая проверка платежей** (полуавтоматический режим)
- 🛡️ **Админ-панель** с управлением через кнопки

### 📱 **Пользовательский интерфейс**
- 🎨 **Интуитивное меню** с кнопками снизу экрана
- 🔄 **Быстрая навигация** между разделами
- 📝 **Простая покупка** в 3 клика
- 📨 **Встроенная поддержка** для связи с админом

## 🚀 Быстрый старт

### Предварительные требования
- Python 3.11 или новее
- Аккаунт Telegram (@BotFather для создания бота)
- Аккаунт Fragment.com (для API ключа)
- Аккаунт CloudTips.ru (для приема платежей)

### Установка за 5 минут

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/your-username/telegram-stars-bot.git
cd telegram-stars-bot

# 2. Установите зависимости
pip install -r requirements.txt

# 3. Настройте файл .env
cp .env.example .env
# Отредактируйте .env файл (см. ниже)

# 4. Запустите бота
python bot.py
